# Dark-Fantasy-Shark-Wizard-III
GP2Team2_HWG_Prototype_ART:
 - Tower model
 - Modular brick models
 - Dark Fantasy Shark Wizard III model
 - Chest model
 - Goblin FPS arms model
 - Beginnings of the main menu UI
 - Skybox

GP2Team2_HWG_Prototype_GP:
 - Player movement
 - Blockout layout of the keep
 - Functional static traps
 - Functional chasing enemies
 - Win condition

How to play:
 - Play at 1920x1080p for the best experience
 - Keyboard: WASD to move, Hold space to jump, mouse to look.
 - Enter the keep and try to reach the top of the tallest tower to reach Dark Fantasy Shark Wizard III's chest (macguffin.)
 - Try to avoid traps and enemies, even one hit is enough to kill you!

GitHub: https://github.com/Neilf65/Dark-Fantasy-Shark-Wizard-III
